March 18, 2021

### Purpose

Notes on working with Vue.js.

### vue-cli

#### Vue app creation

Install and configure `@vue/cli` (Vue.js 3.0)

```node
npm install -g --save @vue/cli
```

Create new app (in ./view)

```node
vue create view
```
